# Coffee-Sales-Dashboard-Excel-Project

Overview

This project presents an interactive Coffee Sales Dashboard built entirely in Microsoft Excel using a coffee sales dataset. The dashboard provides insights into sales performance, customer behavior, coffee product trends, and country-wise sales distribution.

The objective of the project is to transform raw transactional coffee sales data into meaningful business insights using Excel-based data analysis and visualization techniques.

Business Objectives

The dashboard helps answer important business questions such as:
1. Which coffee type generates the highest sales?
2. Which countries contribute the most revenue?
3. Who are the top-performing customers?
4. How do sales vary over time?
5. Which roast type and coffee size perform best?
6. How can businesses identify high-value customers?

Dashboard Components
1. Sales Trend Analysis – Visualizes coffee sales performance over time.
2. Country-wise Sales Analysis – Compares sales contribution across countries.
3. Top Customers Analysis – Identifies high-value customers.
4. Product Performance Analysis – Compares different coffee types such as Arabica, Excelsa, Liberica, and Robusta.
5. Interactive Slicers – Enables filtering by coffee type, roast type, loyalty card, and size.

Key Insights
1. Identified top-performing coffee types among Arabica, Excelsa, Liberica, and Robusta.
2. Analysed sales performance across countries, with the United States contributing the highest sales.
3. Highlighted top customers based on total sales contribution.
4. Examined monthly and yearly sales trends over time.
5. Compared sales performance across roast types and coffee sizes to identify the most preferred combinations.
6. Evaluated customer purchasing patterns and loyalty card usage to identify high-value customers.
